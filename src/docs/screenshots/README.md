# Screenshots for Documentation

Place screenshot images here. These are referenced in `about.md`.

## Required Screenshots

| Filename | Description |
|----------|-------------|
| `overview.png` | Main Rotera interface showing image view and 3D view |
| `fspy-comparison.png` | Side-by-side or diagram showing single vs multi-camera |
| `world-point.png` | A world point marked on a corner/feature |
| `viewpoint.png` | Camera viewpoint with background image visible |
| `image-point.png` | Showing connection between world point and pixel |
| `lines.png` | Lines connecting world points |
| `constraint-coordinate.png` | Coordinate locking UI panel |
| `constraint-distance.png` | Distance constraint between two points |
| `constraint-axis.png` | Axis alignment options/dropdown |
| `constraint-coplanar.png` | Coplanar constraint visualization |
| `interface-views.png` | Overview showing 2D view, 3D view, and viewpoint list |
| `blender-export.png` | Blender with imported Rotera project |

## Recommended Size

- Width: 600-800px
- Format: PNG
- Keep file sizes reasonable (<500KB each)
